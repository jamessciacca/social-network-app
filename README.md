## Description

This is an Express/MongoDB app that allows you to use routes to perform CRUD on users, thoughts, friends, and reactions! Routes were performed using Insomnia!

## Installation

-Git Clone the Repository 
-Open in Your VSCode
-Open Terminal
-Run npm i
-Run npm run start
-Enjoy!
